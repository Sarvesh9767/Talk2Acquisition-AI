# Talk2Acquisition - AI Metadata Assistant

An AI-powered semantic metadata search engine built using Streamlit and Sentence Transformers.

## 🔎 Features

- Semantic search over financial metadata
- Governance-aware queries (Frequency, Owner, Steward, Regulatory Source)
- Similarity scoring with confidence levels
- Vendor and Asset Class filtering
- Chat-style UI

## 🛠 Tech Stack

- Python
- Streamlit
- Sentence Transformers (all-MiniLM-L6-v2)
- Scikit-learn
- Pandas

## 📂 Files

- app.py – Streamlit UI
- search_engine.py – Semantic search logic
- talk2acquisition_master_metadata_v2.csv – Metadata dataset
- requirements.txt – Dependencies

## 🚀 Deployment

Deployed via Streamlit Community Cloud.
